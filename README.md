# Enc-Interativa-teste
Teste técnico para empresa Enc Interativa.

# Como baixar e executar o projeto?
  git clone https://github.com/sabrina-alt/Enc-Interativa-teste.git
  
  cd vue-project
  
  npm install
  
  npm run dev 

# Tecnologias usadas 
  vue.js 3
  
  Sass
  
  HTML
  
  CSS
